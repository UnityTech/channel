Unity SDK for XIAOMI (IAP + Ads) Alpha v0.0.7

Posted on 2017-06-05

Unity SDK for XIAOMI provides an easy way to intergrate with XIAOMI’s IAP and Unity Ads in Unity editor. This is the Alpha v0.0.7 build.

When prompted to update Unity IAP, DO NOT update.

Changelog

  • Fix thread problem under Unity 5.3.
  • Fix IAP Catalog hiding error sections in Product editor GUI.
  • Fix MiProductCatalog.prop not appearing after successfully generating (refresh).
  • Catalog GUI supports expanding foldouts (headers) when clicking on foldout names to expand on Unity 5.5+.
  • Show default label for new product instructing user to populate ID.
  • More log details.

Downloads

  • Unity Package (Demo inlcuded)
  • IAP Doc
  • Ads Doc

Support

If you have any question, feel free to contact us at iapsupport@unity3d.com for IAP and chinaads@unity3d.com for Ads.
