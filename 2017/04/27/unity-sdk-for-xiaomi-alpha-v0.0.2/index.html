Unity SDK for XIAOMI (IAP + Ads) Alpha v0.0.2

Posted on 2017-04-27

Changelog

  • A more stable version.

Unity SDK for XIAOMI provides an easy way to intergrate with XIAOMI’s IAP and Unity Ads in Unity editor. This is the Alpha v0.0.2 build.

You can download the the unity package here and the docs can be found here (IAP) and here (Ads).

If you have any question, feel free to contact us at iapsupport@unity3d.com for IAP and chinaads@unity3d.com for Ads.
